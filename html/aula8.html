DOM - Document Object Model

DOM é tudo aquilo que está no HTML, OU SEJA, todas as tags HTML. DOM facilita que você acesse através do JavaScript esses tipos de elemetos. Os acessos sao feitos através de eventos ou ligaçoes de elementos.

querySelector

É uma simplificação do getElementById, getElementByTagName e getElementByClassName. Ou seja, retorna o primeiro elemento dentro do documeto que foi especificado pelo seletor. O querySelector() aceita os atributos ids, classes e outros parametros de seleção. OBS. querySelector NÃO retorna array.

querySelectorAll

É uma simplificação do getElementById, getElementByTagName e getElementByClassName. O querySelector retorna o primeiro elemento dentro do documento e querySelectorAll retorna uma lista de elementos presentes no documento.
