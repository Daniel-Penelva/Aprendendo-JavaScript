Variáveis: No JS é possivel criar três formas de variáveis. Podemos mudar o tipo da variavel livremente (tipagem fraca).

Tipos de Variáveis: Number, String, Boolean, null, undefined, Object; Podemos verificar o tipo do dado com o operador typeof; Number é um tipo de dado para números; Possui três valores simbólicos: +Infinity, -Infinity e NaN (para operaçoes matematicas que não resulta em números); Não existe um tipo definido para inteiros, todo número é Number;

String: String podem ser escritas com aspas simples e duplas; É possível concatenar String com o +;

Undefined e null: null é um tipo de dado que representa um valor; undefined é um tipo de dado para uma variável com valor não atribuído.
