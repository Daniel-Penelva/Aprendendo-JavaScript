DOM - Document Object Model

DOM

  • Uma tnterface de programação para HTML;
  • Por meio dele temos métodos para acessar a árvore de elementos;
  • DOM fornece uma cópia do HTML;
  • Podemos manipular eventos pelo DOM para afetar o HTML;
  • Acessar o DOM caracteriza-se por identificar um elemento do HTML atraves de métodos;
  • Depois podemos manipulá-los de forma que quisermos;
  • Acessar o DOM é semelhante as regras CSS;
  • Podemos acessar por tags, ids, classes;

Seletores: querySelector e querySelectorAll

  • Com estes seletores podemos acessar os elementos baseados em regras de CSS;
