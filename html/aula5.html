Operadores condicionais

  • São instruções que podem determinar o fluxo de uma aplicação
  • Exemplos, if, else, else if
  • Executa um bloco de nota se a instrução for verdadeira
  • Se for falsa, segue o codigo normalmente
  • O else e o if tem a possibilidade de fazer outra verificação e adicionar mais um código de bloco;

Operadores de Comparação

  • São comumente utilizados para verificação nas estruturas de condição;
  • A partir de uma expressão de comparação podemos obter um true ou um false

Comparação de tipo e valor

  • Esses operadores irao comparar o tipo e o valor
  • Dois outros operadores de comparação existem no JS: === e !==;
  • A função é comparar valor e tipo de dado

Operador Lógico NOT

  • Este operador muda o valor que a expressão retornou;
  • Se recebeu true vira false, se recebeu false vira true;
